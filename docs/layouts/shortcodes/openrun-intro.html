OpenRun is an open source deployment platform for code-first internal tools. OpenRun is the easiest way to declaratively deploy web apps with GitOps, on a single-node or onto a Kubernetes cluster. OpenRun gives code-first apps the platform features usually found in low-code tools like Retool.
